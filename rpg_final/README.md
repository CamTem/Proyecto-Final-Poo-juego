Descripción

Este proyecto es un juego RPG desarrollado en Python utilizando la librería Tkinter para la interfaz gráfica.

El jugador puede crear un personaje, elegir una clase, combatir enemigos, obtener armas, mejorar estadísticas y guardar puntuaciones en un ranking.

El objetivo principal es derrotar al Dragón final.

Características del juego
Interfaz gráfica con Tkinter.
Sistema de combate por turnos.
Clases de personaje:
Guerrero
Mago
Arquero
Asesino
Sistema de inventario.
Armas aleatorias con rarezas.
Recompensas después de cada combate.
Ranking guardado en archivo JSON.
Pantallas separadas usando Frames.

Tecnologías utilizadas
Python 3
Tkinter
JSON

Instrucciones de instalación
Requisitos

Tener instalado:

Python 3

Ejecución

Abrir una terminal en la carpeta del proyecto y ejecutar:

python main.py
